## Reporting a Vulnerability

We take the security of `opendialogum` seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**DO NOT** report security vulnerabilities through public GitHub issues.

### Private Disclosure Process

1.  Please email us at [oshaikh13@gmail.com](mailto:oshaikh13@gmail.com).
2.  In your report, please include:
    -   The version of `opendialogum` (or `gum`) affected.
    -   A description of the vulnerability.
    -   Steps to reproduce the issue.
    -   Any relevant logs or screenshots.

### Response Timeline

-   **Confirmation**: We will acknowledge receipt of your report within 48 hours.
-   **Assessment**: We will assess the issue and provide an estimated timeline for a fix within 1 week.
-   **Resolution**: We will issue a patch and a security advisory once the vulnerability is resolved.

Thank you for helping keep `opendialogum` secure!
